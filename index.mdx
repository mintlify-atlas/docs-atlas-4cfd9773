ElastomerClient is a Ruby gem that provides a one-to-one mapping of the Elasticsearch HTTP API. It gives you direct access to every Elasticsearch endpoint without abstractions getting in the way, so your application code stays in control.

## What ElastomerClient does

ElastomerClient wraps the Elasticsearch HTTP API in idiomatic Ruby. Each major API area maps to a dedicated class, so you can work with indexes, documents, and cluster state through a clean, consistent interface.

<Steps>
  <Step title="Install the gem">
    Add `elastomer-client` to your `Gemfile` and run `bundle install`.
  </Step>
  <Step title="Connect to Elasticsearch">
    Create a client pointing at your Elasticsearch cluster — local or remote.
  </Step>
  <Step title="Work with your data">
    Create indexes, index documents, run searches, and manage your cluster.
  </Step>
</Steps>
